# UI Rules

Este documento define las **reglas de diseño visual y experiencia de usuario** del sitio web del ilustrador.
Todos los agentes de IA que trabajen en el proyecto deben seguir estas reglas para mantener una **identidad visual coherente**.

El estilo del sitio debe reflejar el **estilo artístico del ilustrador**, inspirado en:

* Surf culture
* Skate culture
* Tattoo / line art
* Ilustración artesanal
* Gráfica para tablas de surf, ropa y marcas independientes

Las ilustraciones del artista son la **referencia principal del diseño visual**.

---

# Principios de diseño

## 1. El arte es el protagonista

El objetivo del sitio es **mostrar las ilustraciones**.

Por lo tanto:

* El diseño debe ser **minimalista**
* Mucho **espacio en blanco**
* Evitar UI pesada o demasiado decorativa
* El arte debe ocupar **la mayor parte de la pantalla**

---

## 2. Estética artesanal

El sitio debe transmitir una sensación:

* Artística
* Hecha a mano
* Independiente
* Creativa

Evitar:

* Estética corporativa
* Componentes demasiado "tech"
* Interfaces tipo SaaS

---

## 3. Navegación simple

La navegación debe ser clara y mínima.

Secciones principales:

* Home
* Work
* Categories
* About
* Contact

No agregar navegación compleja.

---

# Layout

## Contenedor principal

Usar ancho máximo:

max-width: 1200px

Centrado en pantalla.

Espaciado amplio entre secciones.

---

## Grids de ilustraciones

Las galerías deben usar **grid responsive**.

Ejemplo:

Desktop
4 columnas

Tablet
2 columnas

Mobile
1 columna

Cada item debe ser:

* Imagen grande
* Hover simple
* Click abre vista ampliada o detalle

---

# Colores

El sitio debe usar **paleta simple** para no competir con las ilustraciones.

## Colores base

Fondo principal

#ffffff

Texto principal

#111111

Color secundario opcional

#555555

Evitar colores fuertes en la UI.

Los colores deben provenir **principalmente de las ilustraciones**.

---

# Tipografía

Usar tipografías simples.

Recomendadas:

Sans serif limpia.

Ejemplos:

* Inter
* Helvetica
* System UI

Reglas:

Titulos grandes
Texto minimalista
Evitar muchas familias tipográficas.

---

# Botones

Los botones deben ser simples.

Ejemplo:

* borde negro
* fondo blanco
* hover negro

Ejemplo visual conceptual

[ Contact ]

Hover

fondo negro
texto blanco

---

# Animaciones

Las animaciones deben ser **mínimas y suaves**.

Permitido:

* Fade in
* Hover en imágenes
* Micro animaciones

Evitar:

* Animaciones pesadas
* Parallax excesivo
* Transiciones largas

---

# Responsive Design

El sitio debe ser **mobile-first**.

Breakpoints sugeridos:

Mobile
0 - 640px

Tablet
640px - 1024px

Desktop
1024px+

En mobile:

* navegación simple
* grids de 1 columna
* imágenes grandes

---

# Accesibilidad

El sitio debe cumplir buenas prácticas básicas:

* imágenes con alt text
* contraste adecuado
* navegación accesible
* links claros

---

# Referencias visuales

Las referencias visuales del estilo están en:

/references/

Los agentes de IA deben usar esas imágenes como inspiración para:

* composición
* ritmo visual
* uso de espacio
* jerarquía

---

# Regla fundamental

El sitio **no debe competir con las ilustraciones**.

El diseño debe ser **silencioso**, permitiendo que el arte sea el centro de atención.
